Proyek Kelompok

Berikut adalah penjelasan umum dari HTML, CSS, Javascript, dan Bootstrap, yang kami gunakan sebagai elemen dari proyek.


HTML

HTML (HyperText Markup Language) adalah bahasa dasar yang digunakan untuk menyusun struktur halaman web. Dengan HTML, kita dapat menentukan elemen-elemen seperti judul, paragraf, gambar, tabel, dan tautan yang akan ditampilkan di browser.

CSS

CSS (Cascading Style Sheets) berfungsi untuk mengatur gaya dan tampilan dari elemen HTML. CSS memungkinkan kita mengubah warna, ukuran font, jarak antar elemen, serta membuat layout halaman menjadi lebih menarik dan teratur.

Bootstrap

Bootstrap adalah framework CSS yang memudahkan pembuatan desain web responsif. Dengan Bootstrap, kita dapat menggunakan komponen siap pakai seperti navbar, tombol, dan grid system tanpa harus menulis kode dari nol, sehingga mempercepat proses pengembangan.

Javascript

JavaScript atau kerap disebut JS adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas pada halaman web. JavaScript memungkinkan halaman web merespon tindakan pengguna, seperti klik tombol atau input data, tanpa perlu memuat ulang halaman.
